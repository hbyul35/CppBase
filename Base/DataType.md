# 자료형  
저장하는 패턴과 유형이 다양해, 필요한 부분만 쓰기위해 자료형은 중요하다.

1. 문자형  
문자 하나당 하나의 ASCII 코드로 표현한다.
2. 정수형    
정수를 표현한다.  
3. 실수형(부동소수형)  
실수를 표현한다.


|구분|자료형|크기|
|:---:|:---:|:---:|
|문자형|char|1byte|
|정수형|short|2byte|
||int|4byte|
||long|8byte|
||long long|8byte|
|실수형|float|4byte|
||double|8byte|
||long double|16byte|
|부울형|bool|1byte|
